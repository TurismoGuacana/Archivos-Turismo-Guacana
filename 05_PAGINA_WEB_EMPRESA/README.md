# Pagina web de Turismo Guacana

Carpeta de trabajo para crear la pagina web de la empresa.

## Estructura

- `00_BRIEF_Y_CONTENIDO`: textos, objetivos, servicios, informacion de contacto y mensajes clave.
- `01_DISENO`: referencias visuales, wireframes, paleta, tipografias y decisiones de interfaz.
- `02_DESARROLLO`: codigo fuente del sitio web.
- `03_RECURSOS_VISUALES`: fotos, logos, iconos, videos y material grafico.
- `04_PUBLICACION`: dominio, hosting, credenciales tecnicas no sensibles, checklist de lanzamiento y respaldos.

## Primeros pendientes

1. Definir objetivo principal de la pagina.
2. Reunir logo, colores, fotos reales y datos de contacto.
3. Escribir las secciones iniciales: inicio, destinos, planes, nosotros y contacto.
4. Elegir tecnologia para construir el sitio.
